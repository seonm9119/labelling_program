📖 사용법

  1. 이미지 폴더: 클릭하여 처리할 이미지들이 있는 폴더 선택
  2. 어노테이션 JSON: 수동 맵핑 결과 JSON 파일 업로드 (템플릿으로 사용)
  3. 물류 OCR 폴더: 클릭하여 물류 OCR JSON 파일들이 있는 폴더 선택
  4. PaddleOCR 폴더: 클릭하여 PaddleOCR JSON 파일들이 있는 폴더 선택
  5. 일괄 처리 시작: 버튼을 클릭하여 자동 처리 시작
  6. 결과 다운로드: 처리 완료 후 ZIP 파일로 다운로드
💡 파일명 매칭: 이미지 파일명과 동일한 이름의 PaddleOCR, 물류OCR JSON 파일을 자동으로 찾아 처리합니다.
예: image1.jpg → image1.json (PaddleOCR), image1.json (물류OCR) → image1.json (출력)
